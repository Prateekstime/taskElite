Project overview:-
This project is a Task & Project Management Dashboard built to help users organize their work efficiently while offering a sleek and modern UI. It features a calendar, task list, project tracker, and achievements panel — all in one centralized interface.

🧩 Tech Stack
      Frontend:

        Next.js – React-based framework for fast and scalable web apps
        
        Tailwind CSS – Utility-first styling for rapid UI development
        
        shadcn/ui – Prebuilt accessible components styled with Tailwind
        
        ApexCharts – Rich data visualization for an engaging experience

Backend:

    MongoDB – NoSQL database for scalable and flexible data storage
    
    Postman – Used to create and test REST APIs

🔐 Authentication
    JWT-based authentication is implemented for admin users
     Other users can currently register and access features without authentication (to be updated in future releases)

🧾 Features
🔧 Admin Panel with secured endpoints using JWT

📆 Sidebar Navigation includes:
    
    Calendar
    
    Tasks
    
    Projects
    
    Achievements

📊 ApexCharts Integration for visually appealing stats and progress tracking

✅ CRUD Operations for:

    Tasks (Create, Read, Update, Delete)
    
    Projects (Create, Read, Update, Delete)
    
    Note: Only admins can perform POST and DELETE operations for security

🚀 Upcoming Improvements
     1. Full user authentication with role-based access
  
     2. Notification system for deadlines
      
     3. Real-time collaboration features
